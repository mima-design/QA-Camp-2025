# QA-Camp-2025

## Prepare enviroment

1. Go to your project folder

```
cd your/project/folder
```

2. Clone repository

```
git clone "page adress"
```

3. Install playwright

```
npm install
```

## Run tests

```
npx playwright test
```

## Run Prettier

You can install VS Code extension: Prettier or write in console:
```
npx prettier --write .
```
